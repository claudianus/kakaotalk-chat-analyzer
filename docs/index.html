미리보기 시작 CSV 위치 팁 자세히 (README)

카카오톡 대화를
한 줄로 리포트.

채팅방에서 CSV 보내기만 하면, Wrapped·차트·키워드가 담긴 공유 링크를 받을 수 있습니다. 대화 원문은 리포트에 넣지 않습니다.

v0.16.1 Node.js 22+ 설치 없이 npx

가장 쉬운 방법 — 경로 생략 (최신 CSV 자동)

npx kcachat@latest

내 PC에만 저장

npx kcachat@latest --local

터미널에 나온 공유 URL을 브라우저에서 열면 됩니다. 최초 1회는 한국어 분석 모델 다운로드로 1~3분 걸릴 수 있습니다.

리포트 미리보기

집계·통계만 담긴 HTML입니다. 참여자 이름은 일부만 보이게 마스킹됩니다.

3단계로 시작

  1. 카카오톡(PC) 채팅방 → 더보기(≡) → 대화 보내기 → CSV 보내기
  2. 터미널(Node.js 22+)에서 위 복사 명령 실행 — 보통 저장 폴더의 가장 최근 CSV를 자동으로 고릅니다
  3. 출력된 링크를 열어 리포트 확인 · 친구에게 공유

CSV는 어디에 저장되나요?

경로를 적지 않아도, 아래 폴더에서 KakaoTalk로 시작하는 CSV를 찾습니다.

OS기본 폴더
Windows 문서\카카오톡 받은 파일
Mac 다운로드 (Downloads)

자주 쓰는 팁

여러 CSV 중 고르기

npx kcachat@latest latest --list로 목록을 보고, --pick 1처럼 순서를 지정할 수 있습니다.

파일 직접 지정

npx kcachat@latest "C:\경로\KakaoTalk_Chat_....csv"처럼 경로를 넣어도 됩니다.

업로드 없이 보기

--local을 붙이면 내 PC 폴더에 index.html만 만들고, 인터넷에 올리지 않습니다.

공유 전에 확인

이름은 마스킹되지만 통계·키워드는 방 분위기가 드러날 수 있습니다. 링크를 보내기 전에 한 번 훑어 보세요.
