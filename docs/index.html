대화는 그대로,
리포트는 안전하게.

KakaoTalk Chat Analyzer는 카카오톡 CSV보내기를 읽어 집계 통계만 담은 단일 HTML 리포트를 만들고, 원하면 가입 없는 임시 호스팅까지 연결합니다. 원문 메시지와 전체 URL은 리포트에 남기지 않습니다.

Node.js 22+ MIT License Cross-platform CLI
저장소 바로가기 설치 · 사용법

한 줄로 시작

# 저장소 클론 후
npm install && npm run build

# 구조 점검 (원문 출력 없음)
node dist/src/cli.js inspect ./chat.csv

# 로컬 리포트
node dist/src/cli.js analyze ./chat.csv -o ./report

# 업로드까지 (기본: BrewPage)
node dist/src/cli.js publish ./chat.csv
Privacy-first

익명·집계 중심

참여자는 별칭으로만 표시되고, 메시지 본문은 통계 계산에만 사용됩니다. 도메인 단위로만 URL을 요약합니다.
