KakaoTalk Chat Analyzer는 카카오톡 CSV보내기를 읽어 한글 프리미엄 리포트(하이라이트·월별·심야 비중 등)를 만들고, 기본은 참여자 부분 마스킹으로 누구인지 감은 오게 보여 줍니다. 원문 메시지와 전체 URL은 HTML에 남기지 않습니다.

실행 (복사)

로컬 npx kakaotalk-chat-analyzer@latest ./보내기.csv --local
업로드 npx kakaotalk-chat-analyzer@latest ./보내기.csv

출력 폴더는 기본 .tmp/kca-report · -o ./report 로 바꿀 수 있어요. inspect·GitHub 설치 등 → README

Privacy-first

부분 마스킹

기본 모드는 이름 앞뒤만 남기고 가운데를 가려, 방 안에서 누구인지 감은 오게 공유할 수 있게 합니다. 완전 별칭 모드도 선택 가능합니다.

Single file

단일 HTML 리포트

배포와 보관이 쉬운 하나의 index.html로 시각화합니다. 브라우저만 있으면 열어볼 수 있습니다.

Publish

임시 공유

BrewPage, TempFile, Cloudflare 등 호스트를 선택해 TTL 기반 임시 링크를 만들 수 있습니다.
