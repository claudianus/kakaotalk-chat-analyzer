카카오톡 대화를
한 줄로 리포트.

채팅방에서 CSV 보내기로 저장한 파일을 넣으면, Wrapped·차트·키워드가 담긴 HTML 리포트 링크를 받을 수 있습니다.

v0.16.1 Node.js 22+
npx kcachat@latest "./KakaoTalk_Chat_보낸파일.csv"

경로는 저장한 CSV 파일 위치로 바꿔 주세요. 터미널에 나온 링크를 브라우저에서 열면 됩니다.

리포트 미리보기

실제 생성 화면 예시입니다. 메시지 원문은 HTML에 넣지 않습니다.

시작하기

  1. 카카오톡 채팅방 → 더보기(≡) → 대화 보내기 → CSV 보내기
  2. 위 명령을 터미널에 붙여 넣고 CSV 경로만 맞춥니다 (Node.js 22+ 필요)
  3. 출력된 링크를 열어 리포트를 확인합니다

팁

PC에만 저장

업로드 없이 폴더에 index.html만 만들려면 끝에 --local을 붙입니다.

진행 표시 끄기

터미널 로그가 거슬리면 --no-progress를 추가합니다. 기본은 진행률 표시 ON입니다.

개인정보

참여자 이름은 기본으로 일부만 보이게 마스킹합니다. 공유 전에 리포트를 한 번 훑어 보세요.
