KakaoTalk Chat Analyzer(CLI 이름 kca)는 카카오톡 CSV 보내기 파일을 읽어 집계 전용 한글 리포트(월별·시간대·키워드·하이라이트 등)를 만듭니다. 기본은 참여자 부분 마스킹이고, 메시지 원문·전체 URL 문자열은 생성된 HTML에 넣지 않습니다.

npx · kcachat Node.js 22+ MIT
저장소 보내기 → 실행 전체 문서

터미널에서 (복사)

로컬
npx kcachat@latest "./KakaoTalk_Chat_보낸파일.csv" --local
업로드
npx kcachat@latest "./KakaoTalk_Chat_보낸파일.csv"

--local이면 PC에만 index.html을 만들고 기본 BrewPage 업로드는 생략합니다. 출력 기본 .tmp/kca-report · 경로 바꾸기 -o ./report. 패키지 전체 이름은 kakaotalk-chat-analyzer와 동일하게 동작합니다. inspect · 다른 호스트 → README

임시 공유

기본은 BrewPage 업로드입니다. TempFile·Cloudflare 등 다른 호스트와 TTL은 README와 kca --help를 참고하세요.

시작하기: 카카오톡에서 CSV 보내기 → 터미널

  1. 카카오톡에서 해당 채팅방 열기 → 우측 상단 더보기(≡)
  2. 대화 보내기 → CSV 보내기로 파일 저장 (파일명은 보통 KakaoTalk_Chat_… 형태)
  3. Node.js 22 이상 설치 후, 저장한 CSV 경로를 넣어 위 명령 실행
  4. 로컬이면 생성 폴더의 index.html을 브라우저로 열기 · 업로드 모드면 터미널에 출력된 링크 확인

이 사이트는 GitHub Pages로 main 브랜치 푸시 시 자동 배포됩니다. 버그·아이디어는 Issues, 기여는 Pull request로 환영합니다. 카카오 공식 제품이 아니며, 보내기 형식이 바뀌면 파싱이 깨질 수 있으니 중요한 원본은 따로 보관하세요.
